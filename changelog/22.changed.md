- Migrated `PgmqBus` from the deprecated `tembo-pgmq-python` package to the official `pgmq` package. The optional extra now installs `pgmq>=1.0.6,<1.1` instead. The import path moves to `from pgmq.async_queue import PGMQueue`; the rest of the API surface is unchanged. Pinned below 1.1 because that release requires PGMQ extension v1.6+ on the database side, which Supabase does not yet ship.
