- Added a pluggable backend layer to `PgmqBus`. Two backends ship: `DirectPgmqBackend` (calls `pgmq.*` directly with prefix-based peer discovery; suitable when bus peers trust each other) and `IsolatedPgmqBackend` (calls `public.bus_*` SECURITY DEFINER wrappers over an asyncpg pool; suitable when peers should be isolated and the channel name is the bus capability). Construct `PgmqBus` with `pgmq=` for the direct backend or `backend=` for any custom or wrapper-based backend.
