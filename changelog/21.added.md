- Added `PgmqBus`, a distributed agent bus backed by PGMQ (PostgreSQL Message Queue) via the official [`pgmq`](https://pypi.org/project/pgmq/) Python client. Each instance broadcasts to peer queues sharing the channel prefix, providing fan-out semantics on top of PGMQ's point-to-point queues. Install with `pip install pipecat-ai-subagents[pgmq]`. A runnable example is included under `examples/distributed/pgmq-handoff/`. The `pgmq` extra is pinned to `>=1.0.6,<1.1` because pgmq 1.1.0 requires PGMQ extension v1.6+ on the database, which Supabase does not yet ship.
